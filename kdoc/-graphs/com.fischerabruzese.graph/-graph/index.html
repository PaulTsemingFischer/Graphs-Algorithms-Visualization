Graph

abstract class Graph<E : Any> : Iterable<E>

Represents a mutable graph data structure. A graph is directed and does not permit negative edge weights.The graph may contain cycles, and is therefore not strictly a DAG. The graph permits Any elements as the vertex type and does not permit nullable types.

Implementations can store vertices in an Adjacency Matrix, Adjacency List, or a different way.

All implimentation of Graphs contain the following algorithms that are either defined here (and are open for optimization in subclasses) or must be defined in its entirety in the specific implementation.

Graph's contain methods for clustering and pathing, including:

  • Dijkstra's Algorithm

  • Breadth First Search

  • Depth First Search

  • HCS (Highly Connected Subgraphs)
